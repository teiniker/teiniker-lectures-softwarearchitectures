Exercise: Selenium Test - Add User
-------------------------------------------------------------------------------

Starten Sie die Web Applikation presentation/MVC-UserForm und überprüfen Sie 
die Funktionalität:

URL: http://localhost:8080/

ID: 7
Username: homer
Password: duffbeer
[AddUser]

=> 7: homer / duffbeer
[Back]

A) SeleniumIDE
Verwenden Sie die SeleniumIDE um einen Test aufzuzeichnen, der diesen manuellen 
Test beliebig oft wiederholt.

B) JUnit Test 
Exportieren Sie den SeleniumIDE Test als Java JUnit Test.
Übernehmen Sie den generierten Code (Teile davon) und fügen Sie diese in die 
Klasse "SeleniumTest" ein => testAddUserHomer()  
Führen Sie diesen Testfall aus.

C) Weiterer JUnit Test 
Fügen Sie einen weiteren Testfall hinzu: testAddUserMarge()
    ID: 8
    Username = "marge"
    Password = "bluehair"
